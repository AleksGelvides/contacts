# Getting Started

### Конфиги

билд скрипт для запуска в Docker: <br>
`./gradlew bootJar`<br>
`cd docker` <br>
`docker-compose build`<br>
`docker-compose up -d` <br>

билд скрипт для запуска локально: <br>
`./gradlew bootJar`<br>
`cd docker` <br>
`docker-compose build bd-dev`<br>
`docker-compose up -d bd-dev` <br>
